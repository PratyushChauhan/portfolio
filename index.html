Evaluating Top-k RAG-based approach for Game Review Generation

IEEE 2024

P. Chauhan, R. K. Sahani, S. Datta, A. Qadir, M. Raj and M. M. Ali — 2024 IEEE International Conference on Computing, Power and Communication Technologies (IC2PCT), Greater Noida, India, pp. 258–263

A study on using RAG + LLaMA 2 13B via LlamaIndex to generate game reviews from public review corpora using top-k semantic retrieval — no fine-tuning required. Demonstrates that context-specific product summaries can be generated purely through RAG over vector stores.
